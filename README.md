# Vanguard - Indian Election Guide

Vanguard is an AI-powered educational platform designed to help Indian citizens understand the election process.

## Features

- **AI Chat Assistant**: Ask questions about voter registration, eligibility, and polling procedures.
- **Election Timeline**: A step-by-step guide to the election lifecycle.
- **Voter Readiness Checklist**: Ensure you're prepared for polling day.

## Tech Stack

- **React + Vite**
- **Tailwind CSS**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Google Gemini API** (AI)

## Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Create a `.env` file and add your `VITE_GEMINI_API_KEY`.
4. Run the development server: `npm run dev`
